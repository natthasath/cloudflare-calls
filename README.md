# 🎉 Cloudflare Calls

Cloudflare Calls is a scalable infrastructure for real-time audio, video, and data applications, utilizing WebRTC to enable low-latency communication across various platforms without the complexities of scaling or regional limitations.

![uptime](https://img.shields.io/badge/uptime-100%25-brightgreen)

### 🚀 Setup

- Create Project

```shell
composer init
```

- Install Package

```shell
composer require vlucas/phpdotenv
```

- Configure Environment

```shell
cp .env.example .env
```

### 🏆 Run

- [http://localhost:8000/](http://localhost:8000/) username : `admin` password : `admin`

```shell
php artisan serve
```
